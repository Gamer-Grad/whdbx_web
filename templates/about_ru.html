О проекте, история

WHDBX (Wormhole Database Extended) - база данных о червоточинах, ВХ-системах, слиперах и всех типах аномалек/комплексов в них. Разработка началась приблизительно в 2015 году, идеологически основанная на проекте evedb.no-ip.org (кто-то, может быть, помнит), автором которого был ЦЕО моей тогдашней корпорации - Determiniks / AkselMJ. Изначальная база по ВХ-системам, дыркам и аномалькам - вся оттуда. Со временем, конечно, по ходу обновлений игры, база пополнялась, однако история помнит, откуда всё пошло.

Идеей нового проекта стала тесная интеграция с ingame-браузером клиента (да, был такой), чтобы сайт можно было очень эффективно использовать прямо из игры с минимумом телодвижений. IGB имел встроенный javascript API (объект CCPEVE), который позволял (при наличии разрешений) автоматически определять текущую систему пилота, корабль, персонажа и копорацию. Все это позволяло, установив стартовой страницей ингейм-браузера этот сайт, просто нажать на иконку запуска браузера на панельке и получить сразу всю информацию по текущей системе - класс, статики, эффект, ссылки на описания аномалек, последние киллы в системе - без единого клика мышкой. Все ссылки на внутриигровые предметы, корабли, корпорации, на персонажей - открывались в стандартном окошечке информации EVE клиента. Поскольку ингейм-браузер был не быстр (тормозил, прямо скажем), весь дизайн был сделан как можно "легче" - минимум эффектов, анимаций, минимум картинок, минимум кликов мышкой.

Затем случилось непоправимое - в 2016 году CCP в очередном апдейте решило удалить из игры ingame-браузер, сославшись на сложность поддержки, списав тем самым плоды все моих усилий и стараний на НЕТ. Часть функциональности появилась в новом API - Crest, но увы, далеко не вся и не так. Определение положения персонажа "лагало" на несколько минут, никаких тебе открытия окошек с инфой в клиенте, и т.д. Таким образом, сайт оставался, конечно, рабочим из обычного браузера, но с половиной функционала попросту обрезанным. Ещё и я тогда стал уделять меньше времени игре, выехал из ВХ, и сайт остался без поддержки и обновлений...

И вот, лето 2017 года, предпринимается попытка "воскресить" былое величие, актуализировать данные по последним обновлениям в области червоточин. Будет попытка использовать очередной новый API от CCP (когда они уже устаканят?) - EVE Swaager Interface. Вроде как он предлагает более тесную интеграцию, нелагучие обновления информации.

На случай если я когда-нибудь всё-таки вконец заброшу EVE Online, все это не должно пропасть. Поэтому весь исходный код проекта выложен в открытый доступ на GitHub - github.com/minlexx/whdbx_web. Pull requests welcome, как говорится. ISK donations слать на персонажа Lexx Min :)
